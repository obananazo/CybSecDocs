# Organisation

It is highly recommended to store any aquired information in organized fashion.

## Folder Structure

An example folder structure for a company might look like this:

```sh
Projects/
└── Acme Company
    ├── EPT (External Penetration Test)
    │   ├── evidence
    │   │   ├── credentials
    │   │   ├── data
    │   │   └── screenshots
    │   ├── logs
    │   ├── scans
    │   ├── scope
    │   └── tools
    └── IPT (Internal Penetration Test)
        ├── evidence
        │   ├── credentials
        │   ├── data
        │   └── screenshots
        ├── logs
        ├── scans
        ├── scope
        └── tools
```
